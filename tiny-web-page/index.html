Tiny web page

This is a tiny web page, consisting of just 3 files:

  1. index.html
  2. style.css
  3. logo.png

We can use this as a place to try out some branching and merging, and to explore when two changes can be merged automatically, and when they can't.

Merge conflicts are nothing to be afraid of!

Let's learn:

  1. what they are;
  2. why they happen;
  3. how to deal with them;
  4. how to avoid them in the first plaice.
